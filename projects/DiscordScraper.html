Project DiscordScraper

Discord scraper for guild members profile picture, username, ID and Bio.

About

DiscordScraper is a simple yet powerful scraper made in Python, that helps
people scrape lots of data about all members in a specific guild/server, really fast.
It was inspired by a JS version made by sertchan on GitHub. My goal was to make the same
project, but with some better features, and more speed, and have the code be easily readable
as Python is way simpler than JS.
