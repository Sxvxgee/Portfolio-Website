Project YouTubeTools

Tools for YouTube, in the form of a command line application, as well as a Discord & Guilded bot.

About

YouTubeTools is a project that comes in 3 different ways: A Discord bot, A Guilded Bot, and a CommandLine script.
The Discord bot part is using a normal Discord Bot instead of a selfbot, as there
is no need for the permissions and things a selfbot does.
The Guilded bot could be outdated at times as I don't use Guilded often, and most people out there
would prefer to use the CommandLine script or the Discord bot version instead.
The CommandLine version is the most advanced version out of all 3, as it's the easiest to code
since there isn't any integration with anything besides the terminal itself and the environment the script is in.
The goal of this project is to make some utilities available to everyone in multiple forms so
more people can access them.
