About

Darkend is one of the first GUI applications made for autofarming on a game inside of a Discord bot called "Dank Memer". Almost all autofarms were TUI based, meaning configuring settings was not simple for people with no coding experience, as those autofarms used JSON files for their settings, and JSON files are extremely sensitive, any illegal character can cause the file not to be readable. And so, Darkend was born.
Darkend allowed it's users to edit the settings directly from the GUI with the click of a button. There was no need to manually edit settings, effectively putting an end to user errors when adjusting the settings. However, that was not it for Darkend. Many other features were added over time that made it truly stand out and beat it's competitors.
Some of it's unique features were the Inventory Scraper, Captcha Solver, Webhook Logging, Data tracking, and so much more!
